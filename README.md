# Site Leonardo Marusso

Este é o projeto para a home principal do domínio leonardomarusso.com.br.

## Tecnologias Usadas

- **React 19**
- **Vite**
- **Tailwind CSS v4**
- **Lucide React** (Ícones)
- **Framer Motion** (Animações)

## Links do Ecossistema

As URLs do projeto (Kit, Toqy, Comunidade, etc.) estão centralizadas em \`src/data/links.ts\` e podem ser facilmente alteradas lá.

## Como Executar

1. Instale as dependências:
   \`\`\`bash
   npm install
   \`\`\`

2. Inicie o servidor de desenvolvimento:
   \`\`\`bash
   npm run dev
   \`\`\`

3. Para criar a versão otimizada de produção:
   \`\`\`bash
   npm run build
   \`\`\`

Este projeto está pronto para fazer deploy direto na Vercel, conectando o repositório e selecionando o framework 'Vite'.
